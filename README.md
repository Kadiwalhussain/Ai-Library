# AI Library

A comprehensive collection of Jupyter notebooks and resources for learning AI/ML with Python. This repository covers essential data science libraries and techniques including NumPy, Pandas, Matplotlib, and working with Web APIs.

## 📚 Contents

### NumPy
Fundamental package for numerical computing in Python.

| Notebook | Description |
|----------|-------------|
| `1_Numpy_array_object.ipynb` | Introduction to NumPy arrays, indexing, slicing, and array manipulation |
| `numerical_operations_on_array.ipynb` | Mathematical operations on arrays |
| `Linear_Algebra.ipynb` | Linear algebra operations with NumPy |
| `stats.ipynb` | Statistical functions and operations |
| `broadcasting.png` | Visual explanation of NumPy broadcasting |

### Pandas
Data manipulation and analysis library.

| Notebook | Description |
|----------|-------------|
| `key_operations_on_dataframes.ipynb` | Essential DataFrame operations (loc, iloc, indexing) |
| `Data_Visualization.ipynb` | Data visualization with Pandas |
| `pandas_prac.ipynb` | Pandas practice exercises |
| `pandas_workshop.ipynb` | Workshop materials for Pandas |
| `Pandas-DataFrame-Notes.pdf` | Reference notes on DataFrames |

### Matplotlib
Visualization library for creating static, animated, and interactive plots.

| Notebook | Description |
|----------|-------------|
| `Matplotlib.ipynb` | Core Matplotlib concepts and architecture |
| `6.2. pyplot.ipynb` | Working with pyplot interface |
| `matplotlib_prac.ipynb` | Practice exercises for plotting |
| `waves_project.ipynb` | Project: Visualizing wave functions |
| `small_project.ipynb` | Mini visualization projects |

### Plotting for Exploratory Data Analysis (EDA)
Techniques for visual data exploration.

| Notebook | Description |
|----------|-------------|
| `1.2. Exploratory Data Analysis..ipynb` | Complete EDA workflow and techniques |

### Introduction to Web APIs
Working with APIs in Python for machine learning applications.

| Resource | Description |
|----------|-------------|
| `BIng_search_api.ipynb` | Bing Search API integration |
| `GCP_speech_to_text_APIs.ipynb` | Google Cloud Platform Speech-to-Text API |

## 🚀 Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook or JupyterLab

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kadiwalhussain/Ai-Library.git
   cd Ai-Library
   ```

2. Install the required packages:
   ```bash
   pip install numpy pandas matplotlib jupyter
   ```

3. For Web API notebooks, install additional dependencies:
   ```bash
   pip install google-api-python-client SpeechRecognition
   ```

4. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## 📖 Learning Path

For beginners, we recommend following this order:

1. **NumPy** - Start with `1_Numpy_array_object.ipynb` to learn array basics
2. **Pandas** - Move to `key_operations_on_dataframes.ipynb` for data manipulation
3. **Matplotlib** - Explore `Matplotlib.ipynb` for visualization fundamentals
4. **EDA** - Apply your skills with `1.2. Exploratory Data Analysis..ipynb`
5. **Web APIs** - Learn to integrate external data sources

## 🔧 Technologies

- **NumPy** - Numerical computing
- **Pandas** - Data analysis and manipulation
